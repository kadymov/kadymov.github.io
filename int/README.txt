Подготовка к собеседованию: JavaScript, React и архитектура

Эта брошюра содержит 25 вопросов и подробных ответов для подготовки к техническому собеседованию на позицию senior frontend разработчика. Вопросы охватывают JavaScript, React, архитектуру, производительность, тестирование и безопасность.

Структура материалов:

Раздел JavaScript включает 7 тем:
- Event Loop, Microtasks и Macrotasks
- Currying в JavaScript
- Let, Const, Var: Scope и Hoisting
- Garbage Collection и оптимизация памяти
- Promises, Async/Await и Generators
- Модель выполнения JavaScript
- Сравнение типизации JS и Java

Раздел React включает 7 тем:
- Reconciliation и Virtual DOM
- Context API vs Redux
- Higher-Order Components
- useEffect и Side Effects
- Конкурентный рендеринг React 18
- Оптимизация ререндеров
- SSR/SSG и React Server Components

Раздел Архитектура включает 6 тем:
- Архитектура состояния приложения
- Микрофронтенд архитектура
- WebSockets и Real-time коммуникации
- Domain-Driven Design в Frontend
- Resilient Frontend паттерны
- Offline-first архитектура

Дополнительные разделы:
- Advanced TypeScript Types
- Стратегии тестирования
- Оптимизация производительности
- Безопасность Frontend приложений

Как использовать материалы:

Первое - последовательное изучение. Начните с JavaScript основ, затем переходите к React и архитектуре.

Второе - практические примеры. Каждый вопрос содержит примеры кода и лучшие практики.

Третье - материал ориентирован на senior уровень и глубокое понимание концепций.

Четвертое - подготовка к интервью. Изучите не только ответы, но и логику рассуждений.

Дополнительные ресурсы для изучения:
- Официальная документация React
- MDN Web Docs для JavaScript
- TypeScript Handbook
- Руководства по веб-производительности

Важно: все ответы актуальны на 2025 год и учитывают современные версии React 18 и выше, TypeScript 5 и выше, Next.js 14 и выше, а также современные браузерные API.
