🔄 Часть 1. Как работают асинхронные функции

Главные участники процесса

📚
Call Stack
Стек вызовов — здесь выполняется синхронный код (LIFO)
🌐
Web APIs
setTimeout, fetch, DOM events — работают вне JS
♻️
Event Loop
Цикл, который связывает очереди со стеком
⚡
Microtask Queue
Promise.then, queueMicrotask — высокий приоритет
📥
Macrotask Queue
setTimeout, setInterval, события — обычный приоритет

📊 Блок-схема выполнения

1️⃣ Синхронный код → Call Stack
Выполняется построчно сверху вниз
⬇️
2️⃣ Async-операции → Web API
setTimeout / fetch / Promise отправляются в браузер
⬇️
⬇️ Promise ⬇️ setTimeout
3a️⃣ Microtask Queue
Promise.then, async/await
3b️⃣ Macrotask Queue
setTimeout, setInterval
⬇️
4️⃣ Event Loop проверяет
Если Call Stack пуст → берёт задачу
⬇️
5️⃣ Возврат в Call Stack → выполнение

🏆 Порядок приоритетов (запомни!)

1
Синхронный код — всё, что в Call Stack прямо сейчас
2
Microtasks — Promise.then, queueMicrotask, await
3
Macrotasks — setTimeout, setInterval, I/O, события
4
Render — отрисовка кадра браузером (после микротасков)
💡 Запомни: СИНХРОН → МИКРО → МАКРО → RENDER

📊 Блок-схема рендеринга

1️⃣ Загрузка HTML/CSS/JS
Браузер получает ресурсы с сервера
⬇️
📄 HTML → DOM
🎨 CSS → CSSOM
⬇️
2️⃣ Render Tree (видимые элементы)
display:none сюда НЕ попадает
⬇️
3️⃣ Layout / Reflow — геометрия
⚠️ Дорогая операция (изменение width, height, позиции)
⬇️
4️⃣ Paint — заливка пикселей
Цвета, фоны, тени, границы
⬇️
5️⃣ Composite — собираем слои на GPU
✅ transform / opacity — самые быстрые анимации

⚙️ Связь с Event Loop (важно!)

  1. JS выполняет таск из очереди (макротаск)
  2. После него выполняются ВСЕ микротаски (Promise)
  3. Браузер решает: пора ли рендерить кадр? (~60 FPS = каждые 16мс)
  4. Если да → вызывается requestAnimationFrame
  5. Затем: Style → Layout → Paint → Composite
  6. Кадр показан → начинается следующая итерация Event Loop
💡 Pipeline: DOM + CSSOM → RT → Layout → Paint → Composite

🚀 Что вызывает что?

🔴
width, height, top, font-size → Layout + Paint + Composite (медленно)
🟡
color, background, visibility → Paint + Composite (средне)
🟢
transform, opacity → только Composite (быстро, GPU) ⚡

🧠 Финальная мнемоника

"СМотрит МАКлер на РЕНДЕР"
Sync → Microtasks → Macrotasks → Render
Рендер: "Дом + Стиль = Дерево, Размер, Краска, Слой"
